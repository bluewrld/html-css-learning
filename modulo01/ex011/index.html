Usando links externos

Você pode acessar o meu repositório no GitHub

Você também pode acessar o canal do Curso em Vídeo gratuito

Para isso utilizei o código:

        
            <a href="https://github.com/bluewrld target="_blank" rel="external">Meu repositório</a>

O target="_blank" serve para abrir o link em uma nova aba do navegador.
