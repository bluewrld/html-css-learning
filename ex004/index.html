Testando carga e Imagens

Abaixo você vai ver uma imagem que está na mesma pasta.

Podemos também carregar imagens que estão em sub-pastas

Também podemos adicionar imagens externas

Para adicionar essas imagens no meu site, utilizei o código:
<img src="diretório da imagem" alt="TEXTO ALTERNATIVO">
esse alt="" serve basicamente para se por algum motivo a imagem não for exibida (erro no caminho, internet fraca, arquivo faltando), o navegador mostra o texto do alt.,

Exemplos:

Para adicionar uma imagem no meu site, que está na mesma pasta de arquivos do site
eu utilizo o código:
<img src="/ex004/HTML5_logo.png" alt="LOGO HTML5">

Para adicionar uma imagem que está em uma sub-pastas
eu utilizo o código:
<img src="/diretorio/imagens/css3_logo.png" alt="LOGO CSS3">

Para adicionar uma imagem de um link externo
eu utilizo o código:
<img src="https://www.studytonight.com/css/resource.v3/icons/subject/logo-javascript.svg" alt="LOGO Javascript"> lembrando que pode ser qualquer link , esse ai foi só um exemplo.
